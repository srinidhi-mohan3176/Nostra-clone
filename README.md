# 🛍️ NOSTRA - Dress Shop Website

NOSTRA is a modern and stylish online dress store where users can explore new arrivals, shop from most-wanted collections, and easily log in or sign up for a personalized experience.

---

## 🚀 Features

### 🆕 New Arrival
- Showcases the latest additions to the store.
- Stylish product cards with image, name, and price.
- Dynamic grid layout that adjusts for desktop and mobile views.

### 🔥 Most Wanted
- Displays top-selling or highly popular items.
- Special styling to highlight trending products.
- Optional labels like “Hot”, “Trending”, “Sold Out”.

### 📦 Collection
- Browse all available product categories.
- Filters and sidebar for types like shirts, suits, combos, etc.
- Grid layout with responsive design for all devices.

### 🔐 Login
- Secure user login form.
- Email & password fields with validation.
- Remember me option.

### 📝 Sign Up
- New users can create an account.
- Includes fields like name, email, password, confirm password.
- Backend-ready structure for authentication integration.

### 📞 Contact Us
- Simple and clean contact form.
- Fields: Name, Email, Message.
- Includes store address, email, and phone (optional).
- Google Maps embed or store image (optional).

---

## 📱 Responsive Design

- 💻 Desktop: Sidebar + Top Navbar layout
- 📱 Mobile: Sidebar collapses; stacked layout (sidebar → search bar → products)
